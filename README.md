# 💧 نظام إدارة المياه (Water Management System)

نظام مالي وإداري متكامل لإدارة محطة مياه (وايتات) مبني على بنية Custom MVC.

## 🏗️ البنية المعمارية

```
water-project/
├── config/             # ملفات الإعداد
│   ├── app.php         # إعدادات التطبيق
│   └── database.php    # إعدادات قاعدة البيانات
├── core/               # الملفات الأساسية (MVC Framework)
│   ├── Database.php    # Singleton PDO Connection
│   ├── Model.php       # Base Model (CRUD)
│   ├── Controller.php  # Base Controller (JSON responses)
│   └── Router.php      # Custom URL Router
├── models/             # نماذج البيانات
├── controllers/        # المتحكمات (API endpoints)
├── services/           # خدمات الأعمال (Fund Service)
├── middleware/          # وسطاء الحماية
│   ├── auth.php        # التحقق من تسجيل الدخول
│   ├── admin.php       # التحقق من صلاحية المدير
│   └── period.php      # منع التعديل في الفترات المغلقة
├── public/             # ملفات الواجهة العامة
│   ├── index.html      # صفحة تسجيل الدخول
│   ├── dashboard.html  # لوحة التحكم الرئيسية
│   ├── api.php         # نقطة دخول API
│   ├── .htaccess       # توجيه Apache
│   ├── css/app.css     # التنسيقات
│   └── js/
│       ├── app.js      # المكتبة الأساسية
│       └── pages.js    # منطق الصفحات
├── db.sql              # هيكل قاعدة البيانات (PostgreSQL)
└── db_mysql.sql        # هيكل قاعدة البيانات (MySQL)
```

## 🚀 التثبيت

### المتطلبات
- PHP 8.0+
- MySQL 5.7+ / MariaDB 10.3+
- Apache مع mod_rewrite

### خطوات التثبيت
1. انسخ المشروع إلى مجلد الخادم
2. أنشئ قاعدة بيانات: `CREATE DATABASE water_management CHARACTER SET utf8mb4;`
3. استورد الجداول: `mysql -u root water_management < db_mysql.sql`
4. عدّل إعدادات الاتصال في `config/database.php`
5. تأكد من تشغيل mod_rewrite في Apache
6. افتح المتصفح على: `http://localhost/water/public/`

### بيانات الدخول الافتراضية
- **المستخدم:** admin
- **كلمة المرور:** admin123

| المستخدمين | إدارة الصلاحيات (Admin only) |

## 🔒 الأمان
- كلمات المرور مشفرة بـ bcrypt
- حماية API بالـ Session
- منع القيم السالبة في العمليات المالية
- قفل الفترات المالية يمنع أي تعديل
- التحقق من الصلاحيات في كل طلب

## 🎨 التصميم
